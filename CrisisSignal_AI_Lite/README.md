# CrisisSignal AI Lite

An AI-powered system that detects crisis-related messages and classifies them into categories.

## Overview
Social media and text platforms often contain emergency messages, but identifying critical situations manually takes time. This project builds an NLP-based system that analyzes user text and identifies whether it represents an emergency situation.

## Features
- Accept user text input
- Classify messages into Accident, Medical, Fire, Flood, Safe/Normal
- Display Predicted category, Emergency level, Confidence percentage, and Suggested action

## Tech Stack
- Frontend: Streamlit
- Backend: Python
- ML Libraries: Pandas, NumPy, Scikit-learn, NLTK
