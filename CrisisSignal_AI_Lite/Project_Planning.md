Project Name:
CrisisSignal AI Lite

Problem Statement:
Social media and text platforms often contain emergency messages, but identifying critical situations manually takes time.

Objective:
Build an AI-powered system that detects crisis-related messages and classifies them into categories.

Target Users:
• General users
• Emergency teams
• Disaster monitoring systems
• Community support groups

Input:
User enters text message

Output:
• Emergency level
• Category
• Confidence score
• Suggested action

Categories:
1. Accident
2. Medical
3. Fire
4. Flood
5. Safe/Normal

Tech Stack:

Frontend:
Streamlit

Backend:
Python

Libraries:
Pandas
NumPy
Scikit-learn
NLTK

Dataset:
Disaster Tweets Dataset
